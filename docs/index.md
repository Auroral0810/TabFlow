# TabFlow 隐私权政策

最后更新日期：2025-02-14

## 1. 信息收集
TabFlow 收集以下信息：
- 浏览器标签页的 URL 和标题
- 用户的 Google 账号邮箱（仅用于数据同步）

## 2. 信息使用
我们收集的信息仅用于：
- 提供标签页管理功能
- 实现跨设备数据同步

## 3. 数据存储
- 本地数据存储在用户的浏览器中
- 同步数据通过 Google 账号加密存储

## 4. 数据共享
我们不会：
- 出售用户数据
- 与第三方共享数据
- 用于营销或广告目的

## 5. 用户权利
用户可以：
- 随时删除所有保存的数据
- 停用数据同步功能
- 完全卸载扩展程序
